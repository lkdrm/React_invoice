# React Invoice Generator

A lightweight, browser-based invoice generator built with React and TypeScript.

Create invoices, calculate VAT, preview professional PDF documents, and keep a local invoice history — all without a backend or user account.

**Live demo:** [Open the application](https://lkdrm.github.io/React_invoice/)

> This project is an MVP intended for learning and portfolio purposes. It is not a substitute for professional accounting, tax, or legal advice.

## Highlights

* Create invoices with supplier, customer, dates, currency, notes, and line items.
* Calculate subtotal, VAT, and final totals automatically.
* Generate PDF invoices with Classic and Minimal templates.
* Save, search, open, duplicate, and delete invoices in local history.
* Configure invoice defaults, VAT rate, currency, template, and invoice-number prefix.
* Support English, Ukrainian, and Czech languages.
* Support light and dark themes.
* Work on desktop and mobile layouts.

## Tech Stack

* React 19
* TypeScript
* Vite
* `@react-pdf/renderer`
* React Context and reducer-based state management
* CSS Modules
* Browser `localStorage`
* Vitest
* GitHub Actions
* GitHub Pages
